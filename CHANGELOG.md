# 변경 이력

## 3.5.0 — 2026-07-15

- 경기 선택 카드 글자 크기를 확대하고 테두리를 4px로 강화
- 선택 카드에 그라데이션, 그림자, 호버·포커스 효과를 적용해 가독성과 디자인 개선
- 모바일에서도 경기 카드 글자와 테두리를 확대
- 관리자 종합 그래프를 단색 파란 막대에서 금·은·동·보라색 누적 막대로 변경
- `전임교원 급여 및 보상`을 `교무팀 - 전임교원 급여 및 보상`으로 변경
- `전임교원 신규임용`을 `교무팀 - 전임교원 신규임용`으로 변경
- 구버전 Google Sheets의 기존 직무명은 새 명칭으로 자동 합산하도록 호환 처리

## 3.4.0 — 2026-07-15

- 관리자 통계 화면 최상단에 선호조사와 기피조사의 누적 Top 4를 각각 별도 카드로 표시
- 누적 Top 4는 최종 1위 4점, 2위 3점, 3위 2점, 4위 1점 기준으로 산정
- 관리자 화면에 기존 설문 결과 전체 초기화 기능 추가
- 전체 초기화 시 `survey_results`, `group_results`, `match_results`의 응답 행을 모두 삭제하고 헤더만 유지
- 오삭제 방지를 위해 `전체삭제` 문구 입력과 확인 체크를 모두 요구

## 3.3.1 — 2026-07-15

- 두 조사 완료 후 마지막 버튼 문구를 `두 조사 모두 마치기를 눌러서 최종 선호조사/기피조사 결과를 확인하세요`로 변경

## 3.3.0 — 2026-07-15

- 조사 완료 후 최대 1,000자의 선택 이유·추가 의견 메모를 선택적으로 입력
- 메모를 Google Sheets `survey_results.selection_reason` 열에 저장
- 관리자 비밀번호로 보호되는 앱 내 관리자 통계 화면 추가
- 선호·기피 필터, 최종 1~4위 횟수, 가중 종합점수 차트, 조별예선 성과, 참여자별 메모 제공
- 관리자 통계 CSV 다운로드 기능 추가
- 최종 1~4위를 가로 배열에서 세로 배열로 변경하고 금·은·동 색상 카드 적용
- 첫 화면에 조별예선 40게임, 결선 16게임 및 누적 데이터 기반 강팀 분산 안내 추가
- Secrets 생성 도구에 `--admin-password` 옵션 추가

## 3.2.0 — 2026-07-15

- 예선 조별리그 진행 중 `A조 예선`, `B조 예선`처럼 현재가 예선조 단계임을 큰 배너로 선명하게 표시
- 예선에서는 `5개 직무 중 1개만 탈락`한다는 안내를 경기 화면과 조별 순위 화면에 더 명확하게 표시
- 첫 화면에 `현재 각 팀의 담당자는 아래 직무만 하는 업무는 아니며, 아래 업무를 포함한 업무를 수행중임` 주의사항 추가
- 4강 종료 직후와 3·4위전 경기 화면에서 `지금은 결승전이 아니라 3·4위전`이라는 안내를 큰 배너로 표시
- 버튼, 본문, 카드, 안내 박스의 글자 크기와 대비를 높여 가독성과 전체 디자인 개선

## 3.1.1 — 2026-07-11

- `전임교원 급여 및 보상`의 소속을 교무팀으로 수정
- 교무팀 4개 업무가 가능하면 A·B·C·D조에 하나씩 배치되도록 동일 소속 분산 규칙에 포함
- 기존 Google Sheets 통계 호환성을 위해 직무명 문자열은 유지
- 교무팀 4개 업무의 조 분산 자동 테스트 추가

## 3.1.0 — 2026-07-11

- 첫 화면에서 선호조사 또는 기피조사 하나를 선택하도록 변경
- 한 조사 완료 후 나머지 조사 계속/참여 종료 선택 추가
- 첫 조사 시작 직전 이름 입력 필수화
- 입력 이름을 `survey_results.participant_name`에 저장
- 우측 상단 `Developed by JK` 크레딧 추가
- 기피조사 편성 시 과거 선호 시드 포트도 동시에 분산
- 현재 참여자 본인의 선호 결과는 본인 기피 시드에서 제외
- 동일 소속의 여러 업무를 가능한 한 서로 다른 조에 배치
- `constants.py`에 편집 가능한 `JOB_UNITS` 추가
- 4강 패자 간 3·4위전 추가
- 최종 1~4위 및 `final_top4_json` 저장
- 한 조사 경기 수를 55회에서 56회로 변경
- 기존 Google Sheet의 접두사형 구버전 헤더 자동 확장
- 시트 열 수가 부족한 경우 자동 resize
- 자동 테스트 20개 및 400개 무작위 편성 스트레스 검증
